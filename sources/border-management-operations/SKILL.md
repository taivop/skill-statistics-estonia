# Border Management Operations

## Use when
- You need official border management operations context.
- You need structured extraction of border policy/operations materials.

## Avoid when
- You need person-level border crossing records.

## Inputs
- Topic scope, period, and operational focus.

## Outputs
- Structured border-operations references and extracted fields.

## Primary endpoints
- Border management: https://www.siseministeerium.ee/en/activities/guaranteed-internal-security/border-management

## Workflow
1. Identify relevant border operations materials.
2. Extract operationally useful fields/indicators.
3. Normalize terminology and period dimensions.
4. Return structured output with citations.

## Human setup (when needed)
- If key data is only in linked reports, guide user through retrieval and continue from provided files.

## Quality checks
- Distinguish policy objectives from operational metrics.
- Keep publication dates and source provenance.
